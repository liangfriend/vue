# 东北小镇餐厅

像素风餐厅经营游戏，参考《吃货大食堂》《Inn Legend》。中国东北小镇，继承父亲的餐厅，招募员工、吸引顾客、赚取金钱。

---

## 游戏背景

你继承了父亲在东北小镇的一间小餐馆。从零开始经营，升级菜谱、招募员工、装修店面，争取成为镇上最好的餐厅。

---

## 一、员工系统

### 属性
| 属性 | 说明 | 影响 |
|-----|------|------|
| 速度 | 移动/服务速度 | 点菜、上菜、收桌耗时 |
| 厨艺 | 烹饪水平 | 菜品品质概率（完美/普通/糊了/夹生） |
| 清洁 | 清洁效率 | 收桌、打扫速度 |
| 魅力 | 对外形象 | 吸引顾客、小费概率 |

### 熟练度
- 每完成一项工作，对应技能熟练度 +1
- 熟练度满 100 可升级该技能 1 级
- 技能等级上限 10

### 薪资
- 按日结算，每天结束扣款
- 基础薪资 + 技能等级加成

### 心情
- 0–100，影响工作效率
- 长时间工作会降低，休息/发奖金可恢复

---

## 二、菜谱系统

### 构成
- 每道菜由多种**基础食材**组成（如：面粉、猪肉、青菜、调料等）
- 食材决定成本，菜谱决定售价

### 星级
- 1–5 星，升星需投入研究点或金钱
- 升星后：售价↑、制作时间↑、品质要求↑

### 菜品品质（由厨艺决定概率）
| 品质 | 售价系数 | 顾客满意度 |
|-----|---------|-----------|
| 完美 | 1.2x | 高 |
| 普通 | 1.0x | 中 |
| 夹生 | 0.5x | 低 |
| 糊了 | 0.2x | 极低 |

---

## 三、食材系统

### 市场
- 每日食材价格随机波动（±15%）
- 可提前囤货，有库存上限

### 基础食材类型
面粉、大米、猪肉、牛肉、鸡蛋、青菜、土豆、番茄、调料、油、盐、糖 等

---

## 四、顾客系统

### 顾客类型
| 类型 | 特点 | 偏好 |
|-----|------|------|
| 学生 | 预算低，量大 | 实惠、快 |
| 上班族 | 午休短，效率优先 | 快、套餐 |
| 文艺青年 | 讲究环境 | 装修、氛围 |
| 家庭客 | 多人桌，消费高 | 儿童友好、大份 |
| 老人 | 清淡、实惠 | 家常菜、安静 |

### 属性词条
- 如：节俭、挑剔、赶时间、喜欢辣、带小孩 等
- 通过算法影响：是否进店、点菜偏好、耐心、小费

### 耐心与心情
- 进店后耐心值随时间下降
- 等待过久心情变差，可能离店或给差评
- 服务好、上菜快可减缓下降

---

## 五、装修系统

### 类型
- 桌椅、地板、墙纸、装饰物
- 部分装饰带功能：饮料机、音响、绿植 等

### 属性
- 每个装修物有吸引力词条
- 如：吸引学生、吸引家庭、提升氛围、提升清洁度
- 组合装修可叠加吸引特定顾客

---

## 六、奇遇系统

### 特殊客人
- 随机出现：美食家、明星、记者 等
- 满足后可获得：名气、奖励、解锁内容

### 节日活动
- 春节、端午、中秋、国庆 等
- 特殊菜单、限时任务、双倍收益

---

## 七、排名系统

### 市排名
- 按日/周/月营业额排名
- 前几名有奖金与称号

### NPC 竞争对手
- 镇上有 3–5 家 NPC 餐厅
- 各有特色，营业额动态变化
- 可视为参照与挑战目标

---

## 像素画资源格式

所有角色、家具、菜品等美术资源使用像素画创建器导出的 JSON 格式：

```json
{
  "width": 16,
  "height": 16,
  "content": [[{"r":255,"g":255,"b":255,"a":255}, ...], ...]
}
```

游戏内通过 PixelSprite 组件渲染。
